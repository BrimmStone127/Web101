Terminology

Internet: A global network of interconnected computers used for transferring information.

World Wide Web: The Web is a system built using internet protocols (primarily HTTP) that allows users to view and share web resource files.

Hypertext: The technology that allows users to view web resources and instantly connects to other web pages.

CLI: Command Line Interface (CLI) is a type of display that uses only text and has no graphics or visuals.

GUI: A Graphical User Interface (GUI) is a interface that uses visuals and styling to display information this is the primary type of interface in use today.

HCI: Human Computer Interaction (HCI) is the study of how humans work alongside computers and finding better ways to design interfaces for easier use.

FTP: File Transfer Protocol (FTP) is a Internet prototcol that is used for moving files between different computers connected to the internet network.

SSH: Secure Shell (SSH) is an internet protocol that allows individuals to securely connect to remote computer systems.

HTTP: Hypertext Transfer Protocol is the Internet protocol used for sending and recieving web resources.

Telnet: Telnet is one of the foundational protocols that lead to modern technologies such as SSH. It allows for virtual connection to remote networks but it is lacking in modern security.

HTML: Hypertext Markup Language (HTML) is the format that the majority of web resource files are written in

CSS: Cascading Syle Sheets (CSS) are a web resource used to create modern styles for HTML documents. Making them easier to read and understand.
